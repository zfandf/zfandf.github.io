使用CSS 修改 input placeholder 占位文本样式

Demo:

样式定义:
/*WebKit和Blink（Safari,Google Chrome, Opera15+）使用伪元素*/
input::-webkit-input-placeholder {color: deeppink; }

/*Mozilla Firefox 4-18使用伪类*/
input:-moz-placeholder {color: deeppink;}

/*Mozilla Firefox 19+ 使用伪元素*/
input::-moz-placeholder {color: deeppink;}

/*IE10使用伪类*/
input:-ms-input-placeholder {color: deeppink;}
